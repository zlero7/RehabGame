GETTING STARTED
1. All icons have their borders adjusted in the sprite editor already and come in sizes 64,128,256,512. (Picto-icons in 32,64,128,256 with 3 color variations each)
2. Backgrounds, grid components, containers, and other similar image sprites follow a similar size scale of 1:2:4:8 and do properly import into projects as is.
3. Buttons come with disabled sprites, highlighted sprites, pressed sprites, and a source image sprite to fully setup an aesthetic button.
4. Demo scenes can be seen in the scenes folder of how setting up these objects may work. 
5. I wrote a simple script to make buttons toggleable on and off, held within the scripts folder. 
6. Need PSD file access for your project? View my PSD version here: https://assetstore.unity.com/packages/2d/gui/space-game-gui-kit-psd-315973
